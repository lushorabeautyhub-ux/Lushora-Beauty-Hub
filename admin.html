🌐 GitHub Pages Hosting Guide

  1. Create a new repository on GitHub (e.g. lushorabeautyhub)
  2. Upload ALL files from this folder: index.html, admin.html, css/, js/, images/
  3. Go to repository Settings → Pages → Source: Deploy from branch → main
  4. Your site will be live at: https://[username].github.io/[repo-name]/
  5. Admin panel will be at: https://[username].github.io/[repo-name]/admin.html

✅ This website is fully compatible with GitHub Pages free hosting. No server or backend required!
